This app uses the MobiledgeX Matching Engine SDK to find and display on a map all of the Cloudlets running the backend app that provides the speed test and computer vision services needed by this app. It uses the following information for the SDK API calls:

appName ${appName}
orgName ${orgName}
appVer ${appVersion}
Operator ${operator}
Region ${region}

These values can be changed in the General Settings of the app.

To learn more about the MobiledgeX SDK, see developers.mobiledgex.com for documentation and a Getting Started guide.
