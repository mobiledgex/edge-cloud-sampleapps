CSV File Format

The CSV file uses comma (',') as separator and needs to have the following format:

Time Date Latitude Longitude
19:36:23 7/26/2018 48.167466 11.345

While Longitude and Latitude have to be of type float (with '.' separator).

  1. The application will send a request to the API for each point to get the information about the network performance.
  2. The application will connect each consecutive point (to draw the according polyline).
  3. Each point will receive a score to determine its color.
  4. The polylines will be drawn.
