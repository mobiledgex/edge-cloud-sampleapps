This app uses the MobiledgeX Matching Engine SDK to find the closest Triton Inference Server app instance for this client to connect to. it uses the following information for the SDK API calls:

appName ${appName}
orgName ${orgName}
appVer ${appVersion}
Carrier ${carrier}
Region ${region}

These values can be changed in the General Settings of the app.

To learn more about the MobiledgeX SDK, see developers.mobiledgex.com for documentation and a Getting Started guide.

Source code for this app is available on Github.
